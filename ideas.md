# Идеи дизайна: AI-микшер Яндекс Музыки

## Концепция 1: «Тёмная студия» (Dark Studio)

<response>
<text>
**Design Movement:** Dark Brutalism + Audio Engineering Aesthetic

**Core Principles:**
1. Тёмный, почти чёрный фон с акцентами нефритово-зелёного цвета — как экраны профессионального аудиооборудования
2. Монохромная типографика с редкими яркими акцентами для статусов и кнопок действий
3. Визуализация аудиоволн и осциллографов как декоративные элементы
4. Жёсткая сетка с острыми углами, без скруглений

**Color Philosophy:**
- Фон: `#0A0A0A` (почти чёрный)
- Акцент: `#00FF88` (нефритово-зелёный, как на старых мониторах)
- Вторичный: `#1A1A2E` (тёмно-синий для карточек)
- Текст: `#E0E0E0` (светло-серый)
- Статус «кэшируется»: `#FFB800` (янтарный)

**Layout Paradigm:**
Вертикальный список с горизонтальными карточками треков. Левая панель — плейлисты, правая — треки. Нижняя панель — плеер/микшер.

**Signature Elements:**
1. Анимированные волны аудио (CSS animation) как фоновый элемент
2. Индикаторы уровня звука (VU-meters) для треков в процессе анализа
3. Монохромные иконки с резкими линиями

**Interaction Philosophy:**
Каждое нажатие — чёткий отклик. Анимации быстрые и функциональные, не декоративные.

**Animation:**
- Появление карточек: slide-in снизу с задержкой 50ms между элементами
- Прогресс кэширования: заполнение полосы прогресса слева направо
- Переход между страницами: fade + slight scale

**Typography System:**
- Заголовки: `JetBrains Mono` (моноширинный, технический)
- Тело: `Inter` (читаемый, нейтральный)
- Акценты: `Space Grotesk` (геометрический)

## Концепция 2: «Ночной клуб» (Neon Noir)

<response>
<text>
**Design Movement:** Neon Noir + Lo-Fi Aesthetic

**Core Principles:**
1. Глубокий фиолетово-чёрный фон с неоновыми акцентами розового и синего
2. Размытые градиентные пятна (glow effects) как источники «света»
3. Полупрозрачные карточки с backdrop-blur (стекло)
4. Асимметричная компоновка с крупными заголовками

**Color Philosophy:**
- Фон: `#0D0D1A` (тёмно-фиолетовый)
- Акцент 1: `#FF2D78` (неоново-розовый)
- Акцент 2: `#00D4FF` (неоново-синий)
- Карточки: `rgba(255,255,255,0.05)` с blur
- Текст: `#FFFFFF`

**Layout Paradigm:**
Полноэкранный мобильный интерфейс с нижней навигацией. Главный экран — большая обложка текущего плейлиста с треками внизу. Свайп для перехода между плейлистами.

**Signature Elements:**
1. Glow-эффекты вокруг активных элементов
2. Glassmorphism карточки с цветными бордерами
3. Анимированный градиентный фон, медленно меняющийся

**Interaction Philosophy:**
Плавные, «жидкие» переходы. Каждый элемент реагирует на касание мягким свечением.

**Animation:**
- Фон: медленная анимация градиента (10s loop)
- Карточки: появление с blur-in эффектом
- Кнопки: pulse glow при нажатии

**Typography System:**
- Заголовки: `Syne` (жирный, геометрический)
- Тело: `DM Sans` (современный, читаемый)
</text>

## Концепция 3: «Минималистичный плеер» (Zen Player)

<response>
<text>
**Design Movement:** Japanese Minimalism + Swiss Grid

**Core Principles:**
1. Светлый фон с большим количеством воздуха и пространства
2. Акцент на типографике и иерархии информации
3. Один основной акцентный цвет (тёмно-красный/бордо) для действий
4. Строгая сетка с выравниванием по базовой линии

**Color Philosophy:**
- Фон: `#F5F0EB` (тёплый кремовый)
- Акцент: `#C0392B` (бордо)
- Карточки: `#FFFFFF` с тонкой тенью
- Текст: `#1A1A1A` (почти чёрный)
- Вторичный текст: `#888888`

**Layout Paradigm:**
Список плейлистов как горизонтальный скролл вверху, треки — вертикальный список. Минимум элементов на экране, максимум информации.

**Signature Elements:**
1. Тонкие разделители между элементами
2. Большие обложки альбомов как основной визуальный элемент
3. Анимация прогресса кэширования как тонкая линия под треком

**Interaction Philosophy:**
Спокойные, предсказуемые анимации. Никаких сюрпризов — только чёткие, понятные действия.

**Animation:**
- Переходы: простой fade (200ms)
- Загрузка: скелетон-экраны вместо спиннеров
- Прогресс: тонкая линия заполняется снизу вверх

**Typography System:**
- Заголовки: `Playfair Display` (элегантный, с засечками)
- Тело: `Source Sans 3` (читаемый, нейтральный)
</text>
