# Яндекс Микшер — Инструкция по запуску

## Что нужно запускать каждый раз после включения Mac

---

## Шаг 1 — Открыть Terminal

Нажмите **Command + Пробел**, введите `Terminal`, нажмите Enter.

---

## Шаг 2 — Запустить бэкенд-сервер

Вставьте в Terminal и нажмите Enter:

```bash
cd ~/Desktop/yandex_mixer && source venv/bin/activate && uvicorn main:app --host 0.0.0.0 --port 8000 --reload
```

**Ждите, пока появится:**
```
Успешный вход в аккаунт: kseniiazhirnenko
INFO: Uvicorn running on http://0.0.0.0:8000
```

> Это окно Terminal нельзя закрывать — сервер должен работать всё время.

---

## Шаг 3 — Открыть второе окно Terminal

Нажмите **Command + T** (новая вкладка) или **Command + N** (новое окно).

---

## Шаг 4 — Запустить ngrok-туннель

Вставьте во второе окно Terminal и нажмите Enter:

```bash
ngrok http --url=aortal-unoxygenized-herma.ngrok-free.dev 8000
```

**Ждите, пока появится:**
```
Forwarding  https://aortal-unoxygenized-herma.ngrok-free.dev -> http://localhost:8000
```

> Это окно тоже нельзя закрывать.

---

## Шаг 5 — Открыть приложение на iPhone

Откройте Safari на iPhone и перейдите по ссылке вашего опубликованного сайта на Manus.

Или откройте иконку **Яндекс Микшер** на экране домой (если установили как приложение).

Должно появиться **ONLINE** и список плейлистов.

---

## Если что-то пошло не так

| Проблема | Решение |
|---|---|
| Пишет OFFLINE / сервер недоступен | Проверьте, что оба Terminal окна открыты и работают |
| ngrok выдаёт ошибку "already online" | Туннель уже запущен, всё в порядке — откройте приложение |
| Сервер не запускается | Убедитесь, что виртуальное окружение активировано (`source venv/bin/activate`) |
| Плейлисты не загружаются | Откройте в Safari: `https://aortal-unoxygenized-herma.ngrok-free.dev/api/status` — должен быть JSON |

---

## Как остановить всё

1. В окне с uvicorn: нажмите **Control + C**
2. В окне с ngrok: нажмите **Control + C**
